## 控制流分析

控制流分析（CFA， Control Flow Analysis）在 TypeScript 文档中只是简单的提了一下，但是这是一个非常重要的概念。

控制流分析是 TypeScript 编译器的一种机制。它会顺着你的代码逻辑（如 if、else、switch、return、throw）走一遍，记录下变量在每一个特定点上最精确的类型。
