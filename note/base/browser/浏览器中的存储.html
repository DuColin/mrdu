cookie вҖӢ

HTTP cookieйҖҡеёёиў«з§°дёәcookieпјҢз”ЁдәҺеңЁе®ўжҲ·з«ҜеӯҳеӮЁдјҡиҜқдҝЎжҒҜгҖӮд№ҹе°ұжҳҜиҜҙcookieжҳҜжөҸи§ҲеҷЁжҸҗдҫӣзҡ„еҠҹиғҪпјҢе®ғжҳҜеӯҳеӮЁеңЁжөҸи§ҲеҷЁдёӯзҡ„зәҜж–Үжң¬пјҢеңЁchorome devtoolsдёӯпјҢжҲ‘д»¬еҸҜд»ҘиҪ»жҳ“зҡ„зңӢеҲ°жөҸи§ҲеҷЁеӯҳеӮЁзҡ„cookieгҖӮ

cookieжҳҜжҖҺд№Ҳе·ҘдҪңзҡ„пјҹ вҖӢ

йҖҡеёёcookieжҳҜжңҚеҠЎеҷЁеҸ‘йҖҒеҲ°з”ЁжҲ·жөҸи§ҲеҷЁе№¶жң¬ең°дҝқеӯҳзҡ„дёҖе°Ҹеқ—ж•°жҚ®пјҢе®ғдјҡеңЁжөҸи§ҲеҷЁдёӢж¬Ўеҗ‘еҗҢдёҖжңҚеҠЎеҷЁеҶҚж¬ЎеҸ‘иө·иҜ·жұӮжҳҜиў«жҗәеёҰе№¶еҸ‘йҖҒеҲ°жңҚеҠЎеҷЁдёҠзҡ„гҖӮжіЁж„ҸпјҢcookie жҳҜдёҺзү№е®ҡеҹҹз»‘е®ҡзҡ„гҖӮ

еҪ“зҪ‘йЎөеҸ‘иө·HTTP/HTTPSиҜ·жұӮж—¶пјҢжөҸи§ҲеҷЁдјҡж №жҚ®и§„еҲҷжЈҖжҹҘжҳҜеҗҰе…·жңүеҜ№еә”зҡ„cookieпјҢеҰӮжһңжңүеҲҷиҮӘеҠЁж·»еҠ еҲ°Request Heahersдёӯзҡ„cookieеӯ—ж®өдёӯпјҢиҝҷжҳҜжөҸи§ҲеҷЁиҮӘеҠЁеё®жҲ‘д»¬еҒҡзҡ„гҖӮ

cookieйҖӮеҗҲеӯҳеӮЁд»Җд№Ҳж•°жҚ®пјҹ вҖӢ

cookieдёӯеӯҳеӮЁзҡ„ж•°жҚ®пјҢжҜҸж¬ЎйғҪдјҡиў«жөҸи§ҲеҷЁиҮӘеҠЁж·»еҠ еҲ°http/httpsиҜ·жұӮеӨҙдёӯгҖӮеҰӮжһңиҜҙиҝҷдәӣж•°жҚ®дёҚжҳҜжҜҸж¬ЎиҜ·жұӮйғҪйңҖиҰҒжҗәеёҰзҡ„пјҢйӮЈд№ҲеҶ—дҪҷзҡ„ж•°жҚ®е°ұдјҡеўһеҠ зҪ‘з»ңејҖй”ҖпјҢдҪҶжҳҜеҰӮжһңиҝҷдәӣж•°жҚ®жҳҜжҜҸж¬ЎиҜ·жұӮйғҪиҰҒеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁпјҢеҲҷжөҸи§ҲеҷЁзҡ„иҮӘеҠЁеӨ„зҗҶе°ұе…ҚеҺ»дәҶејҖеҸ‘иҖ…жүӢеҠЁеӨ„зҗҶзҡ„йә»зғҰгҖӮ

еӣ жӯӨпјҢеҸӘжңүйӮЈд№ҲвҖңжҜҸж¬ЎиҜ·жұӮйғҪйңҖиҰҒеҸ‘йҖҒз»ҷжңҚеҠЎз«Ҝзҡ„ж•°жҚ®вҖқжүҚйҖӮеҗҲж”ҫеңЁcookieдёӯгҖӮйҖҡеёёжқҘи®Ід»ҘдёӢдёүдёӘж–№йқўзҡ„ж•°жҚ®йҖӮеҗҲиў«ж”ҫеҲ°cookieдёӯпјҡ

  • дјҡиҜқзҠ¶жҖҒз®ЎзҗҶгҖӮеҰӮдҝқжҢҒз”ЁжҲ·зҷ»еҪ•зҠ¶жҖҒзӯүгҖӮ
  • дёӘжҖ§еҢ–и®ҫзҪ®гҖӮеҰӮз”ЁжҲ·иҮӘе®ҡд№үзҡ„и®ҫзҪ®гҖҒдё»йўҳзӯүгҖӮ
  • жөҸи§ҲеҷЁиЎҢдёәи·ҹиёӘгҖӮеҰӮи·ҹиёӘеҲҶжһҗз”ЁжҲ·иЎҢдёәзӯүгҖӮ

cookieзҡ„дҪҝз”ЁйҷҗеҲ¶ вҖӢ

иҷҪз„¶иҜҙRFC 2965 жҢҮе®ҡдёҚеә”жңүcookieзҡ„й”®/еҖјзҡ„й•ҝеәҰйҷҗеҲ¶пјҢе№¶йј“еҠұж”ҜжҢҒд»»ж„ҸеӨ§зҡ„cookieгҖӮдҪҶжҳҜдёәдәҶйҳІжӯўcookieзҡ„ж»Ҙз”ЁпјҢйҖ жҲҗжҖ§иғҪзҡ„жөӘиҙ№пјҢеҗ„дёӘжөҸи§ҲеҷЁйғҪеҜ№cookieзҡ„дҪҝз”ЁеҒҡеҮәдәҶдёҖдәӣйҷҗеҲ¶пјҢдҪҶжҳҜиҝҷдәӣйҷҗеҲ¶еҚҙе№¶дёҚз»ҹдёҖгҖӮз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮBrowser Cookie LimitsиҝҷдёӘзҪ‘з«ҷжөӢиҜ•cookieзҡ„йҷҗеҲ¶гҖӮдёӢйқўжҳҜеңЁеҗ„дёӘжөҸи§ҲеҷЁиғҪеӨҹе®үе…ЁдҪҝз”Ёcookieзҡ„йҷҗеҲ¶пјҡ

  • жҜҸдёӘеҹҹеҗҚдёӢзҡ„cookieдёҚиҰҒи¶…иҝҮ4093 bytesпјҲеӨ§зәҰ4KBпјүпјҢиҝҷж„Ҹе‘ізқҖжҜҸжқЎcookieзҡ„еӨ§е°Ҹжӣҙе°ҸдәҶгҖӮ
  • жҜҸдёӘеҹҹеҗҚдёӢиҮіеӨҡеӯҳеӮЁ30 ~ 50 жқЎcookieпјҲиҝҷдёӘиҜҙжі•е№¶дёҚз»ҹдёҖпјҢд№ҹжІЎжүҫеҲ°жөҸи§ҲеҷЁе®ҳж–№иҜҙжҳҺж–ҮжЎЈпјүгҖӮ

cookieзҡ„жһ„жҲҗ вҖӢ

cookieеңЁжөҸи§ҲеҷЁдёӯжҳҜд»ҘдёӢеҸӮж•°жһ„жҲҗзҡ„пјҡ

  • nameпјҡcookieзҡ„е”ҜдёҖж ҮиҜҶпјҢcookieеҗҚдёҚеҢәеҲҶеӨ§е°Ҹе§җгҖӮ

  • value:пјҡеӯҳеӮЁеңЁcookieдёӯзҡ„еӯ—з¬ҰдёІеҖјпјҢе®ғдёҺnameз»„жҲҗй”®еҖјеҜ№гҖӮ

  • domainпјҡcookieжңүж•Ҳзҡ„еҹҹпјҢиҝҷдёӘеҖјеҸҜд»ҘеҢ…еҗ«еӯҗеҹҹпјҲwww.baidu.comпјүпјҢд№ҹеҸҜд»ҘдёҚеҢ…еҗ«еӯҗеҹҹпјҲ.baidu.comпјүпјҢеҰӮжһңдёҚжҳҺзЎ®и®ҫзҪ®пјҢеҲҷй»ҳи®Өи®ҫзҪ®cookieзҡ„еҹҹгҖӮ

  • pathпјҡиҜ·жұӮзҡ„URLеҢ…еҗ«иҝҷдёӘи·Ҝеҫ„жүҚдјҡжҠҠcookieеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁгҖӮ

  • expiresпјҡcookieзҡ„иҝҮжңҹж—¶й—ҙпјҢиЎЁзӨәдҪ•ж—¶еҲ йҷӨcookieзҡ„ж—¶й—ҙжҲіпјҲеҪ“и¶…иҝҮиҝҷдёӘж—¶й—ҙд№ӢеҗҺе°ұдёҚдјҡиў«еҸ‘йҖҒеҲ°жңҚеҠЎеҷЁдәҶпјүгҖӮй»ҳи®Өжғ…еҶөеҢ–пјҢжөҸи§ҲеҷЁеӣһиҜқз»“жқҹеҗҺе°ұдјҡеҲ йҷӨcookieпјӣдҪҶжҳҜд№ҹеҸҜд»Ҙи®ҫзҪ®еҲ йҷӨcookieзҡ„ж—¶й—ҙпјҢе®ғжҳҜGMTж јејҸж—¶й—ҙпјҲеҸҜд»ҘйҖҡиҝҮnew Date().toGMTString()иҺ·еҸ–пјүпјҢиҝҷж ·еҚідҪҝе…ій—ӯдәҶжөҸи§ҲеҷЁcookieд№ҹдјҡдҝқеӯҳеңЁз”ЁжҲ·жңәеҷЁдёҠпјҢжҠҠж—¶й—ҙи®ҫзҪ®дёәиҝҮеҺ»зҡ„ж—¶й—ҙдјҡз«ӢеҚіеҲ йҷӨcookieгҖӮ

  • secureпјҡи®ҫзҪ®е®үе…Ёж Үи®°еҗҺпјҢеҸӘеңЁдҪҝз”ЁSSLе®үе…ЁиҝһжҺҘзҡ„жғ…еҶөдёӢжүҚдјҡжҠҠcookieеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁгҖӮй»ҳи®Өжғ…еҶөдёӢ,cookieдёҚдјҡеёҰsecureйҖүйЎ№пјҢжүҖд»ҘжӯӨж—¶ж— и®әжҳҜHTTPеҚҸи®®иҝҳжҳҜHTTPеҚҸи®®пјҢcookieйғҪдјҡиў«еҸ‘йҖҒеҲ°жңҚеҠЎеҷЁгҖӮ

  • httpOnly: иҝҷдёӘйҖүйЎ№з”ЁжқҘи®ҫзҪ®cookieжҳҜеҗҰиғҪеӨҹйҖҡиҝҮjsи®ҝй—®гҖӮhttpOnlyеҸҜд»Ҙз”ұжңҚеҠЎз«Ҝи®ҫзҪ®пјҢд№ҹеҸҜд»Ҙз”ұе®ўжҲ·з«Ҝи®ҫзҪ®пјҢдҪҶеҸӘиғҪеңЁжңҚеҠЎз«ҜиҜ»еҸ–пјҢеӣ дёәи®ҫзҪ®дәҶhttpOnlyеҗҺж— жі•йҖҡиҝҮJSи®ҝй—®пјҲиҜ»еҸ–гҖҒдҝ®ж”№гҖҒеҲ йҷӨпјүгҖӮ

йҰ–е…Ҳcookieдёӯзҡ„name/valueй”®еҖјеҜ№жҳҜеҝ…йЎ»зҡ„пјҢиҝҷдёӘеҸӮж•°йҖҡиҝҮвҖңеҲҶеҸ·еҠ з©әж јвҖқйҡ”ејҖпјӣе…¶ж¬ЎжөҸи§ҲеҷЁеҢәеҲҶcookieжҳҜеҗҰзӣёеҗҢж—¶пјҢйңҖиҰҒеҗҢж—¶еҲӨж–ӯзӣёеҗҢзҡ„nameгҖҒdomainгҖҒpathе’ҢsecureпјҢиҝҷеңЁдҝ®ж”№е’ҢеҲ йҷӨcookieж—¶еҫҲйҮҚиҰҒгҖӮз®ҖеҚ•зҡ„cookieж јејҸзӨәдҫӢеҰӮдёӢпјҡ

дёәдәҶз»•иҝҮжөҸи§ҲеҷЁеҜ№жҜҸдёӘеҹҹж•°зҡ„жҳҫзӨәпјҢжңүдәӣејҖеҸ‘иҖ…жҸҗеҮәдәҶеӯҗcookieзҡ„жҰӮеҝөгҖӮеӯҗcookieзҡ„жң¬иҙЁжҳҜжҠҠеӨҡдёӘеҗҚ/еҖјеҜ№жӢјжҺҘпјҢдҪңдёәеҚ•дёӘcookieзҡ„еҖјгҖӮеҮӯеҖҹзҡ„ж јејҸзұ»дјјдәҺжҹҘиҜўеӯ—з¬ҰдёІпјҢдҫӢеҰӮпјҡ

http
set-cookie: name=name1=value1&name2=value2&name3=value3; expire=expiration_time; secure; httpOnly;

cookieзҡ„зј–з Ғ вҖӢ

cookieжҳҜдёҖдёӘеӯ—з¬ҰдёІпјҢе№¶дё”иҜҘеӯ—з¬ҰдёІдёӯзҡ„йҖ—еҸ·гҖҒеҲҶеҸ·гҖҒз©әж јйғҪиў«еҪ“еҒҡдәҶзү№ж®Ҡз¬ҰеҸ·гҖӮеӣ жӯӨcookieдёӯзҡ„nameе’ҢvalueжҳҜиў«зј–з Ғзҡ„гҖӮ

  • еңЁиҺ·еҸ–cookieж—¶пјҢйңҖиҰҒеҜ№nameе’ҢvalueиҝӣиЎҢи§Јз ҒгҖӮ

    javascript
    const name = decodeURIComponent(`cookie-name`);
const value = decodeURIComponent(`cookie-value`);

  • еңЁеҶҷе…Ҙcookieж—¶пјҢд№ҹйңҖиҰҒе…ҲеҜ№nameе’ҢvalueиҝӣиЎҢзј–з ҒгҖӮ

    javascript
    document.cookie =`${encodeURIComponent(name)}=${encodeURIComponent(value)}; expire=${expration_time}; secure`;

cookieзҡ„и®ҫзҪ® вҖӢ

  • жңҚеҠЎз«Ҝи®ҫзҪ®гҖӮж— и®әдҪ иҜ·жұӮзҡ„жҳҜиө„жәҗж–Үд»¶иҝҳжҳҜеҸ‘йҖҒдёҖдёӘajaxиҜ·жұӮпјҢжңҚеҠЎз«ҜйғҪдјҡиҝ”еӣһдёҖдёӘresponseпјҢеңЁresponse headerдёӯзҡ„set-cookieеӯ—ж®өжҳҜдё“й—Ёз”ЁжқҘи®ҫзҪ®cookieзҡ„гҖӮдҪҶжҳҜдёҖдёӘset-cookieеӯ—ж®өеҸӘиғҪи®ҫзҪ®дёҖдёӘcookieпјҢиҰҒжғіи®ҫзҪ®еӨҡдёӘcookieпјҢе°ұиҰҒж·»еҠ еӨҡдёӘset-cookieеӯ—ж®өгҖӮ

иҜ»еҸ–cookie вҖӢ

BOMжҸҗдҫӣдәҶdocument.cookieзҡ„еұһжҖ§иҜ»еҸ–еҪ“еүҚеҹҹдёӢзҡ„жүҖжңүcookieгҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢжӯӨж—¶document.cookieиҝ”еӣһзҡ„д»…д»…жҳҜжүҖжңүcookieзҡ„name/valueз»„жҲҗзҡ„еӯ—з¬ҰдёІпјҢе№¶з”Ёе®ғд»¬дҪҝз”ЁвҖңеҲҶеҸ·еҠ з©әж јвҖқжӢјжҺҘгҖӮ

иҰҒжғідҝ®ж”№cookieзҡ„еҖјпјҢеҸӘйңҖиҰҒжҠҠеҗҢдёҖдёӘcookeйҮҚж–°иөӢеҖје°ұеҘҪпјҢд№ҹе°ұжҳҜжҠҠеҗҚз§°гҖҒеҹҹгҖҒи·Ҝеҫ„гҖҒе®үе…ЁйҖүйЎ№зӣёеҗҢзҡ„cookieзҡ„еҖји®ҫзҪ®дёәдёҖдёӘж–°еҖје°ұеҘҪгҖӮ

еҲ йҷӨcookie вҖӢ

жІЎжңүзӣҙжҺҘеҲ йҷӨе·Іжңүcookieзҡ„ж–№жі•пјҢеӣ жӯӨиҰҒжғіеҲ йҷӨдёҖдёӘcookieпјҢеҸӘйңҖиҰҒеҶҚж¬ЎжҠҠеҗҢдёҖдёӘcookieпјҲеҗҚз§°гҖҒеҹҹгҖҒи·Ҝеҫ„гҖҒе®үе…ЁйҖүйЎ№зӣёеҗҢпјүзҡ„expireйҖүйЎ№и®ҫзҪ®дёәдёҖдёӘиҝҮжңҹзҡ„ж—¶й—ҙе°ұеҘҪгҖӮ

е·Ҙе…·еҮҪж•° вҖӢ

еңЁjavascriptдёӯиҜ»еҶҷcookieдёҚжҳҜеҫҲзӣҙи§ӮпјҢеӣ жӯӨеҸҜд»ҘйҖҡиҝҮдёҖдёӘе·Ҙе…·еҢ–еҮҪж•°жқҘиҫ…еҠ©зӣёе…іж“ҚдҪңгҖӮдёӢйқўжҳҜдёҖдёӘеҢ…еҗ«дәҶиҜ»гҖҒеҶҷгҖҒеҲ йҷӨж“ҚдҪңзҡ„е·Ҙе…·еҮҪж•°зҡ„зӨәдҫӢгҖӮ

javascript
class CookieUtil {
    // cookieзҡ„иҜ»еҸ–
    static get(name) {
        let cookieName = `${encodeURIComponent(name)}=`,
            cookieStart = document.cookie.indexOf(cookieName),
            cookieValue = null;
        if (cookieStart > -1) {
            let cookieEnd = document.cookie.indexOf(";", cookieStart);
            if (cookieEnd == -1) {
                cookieEnd = document.cookie.length;
            }
            cookieValue = decodeURIComponent(document.cookie.substring(cookieStart + cookieName.length, cookieEnd));
        }
        return cookieValue;
    }

    // cookieзҡ„дҝ®ж”№е’Ңж·»еҠ 
    static set(name, value, expires, path, domain, secure) {
        let cookieText = `${encodeURIComponent(name)}=${encodeURIComponent(value)}`;
        if (expires instanceof Date) {
            cookieText += `; expires=${expires.toGMTString()}`;
        }
        if (path) {
            cookieText += `; path=${path}`;
        }
        if (domain) {
            cookieText += `; domain=${domain}`;
        }
        if (secure) {
            cookieText += "; secure";
        }
        document.cookie = cookieText;
    }

    // cookieзҡ„еҲ йҷӨ
    static unset(name, path, domain, secure) {
        CookieUtil.set(name, "", new Date(0), path, domain, secure);
    }
}

Web Storage вҖӢ

Web StorageжңҖж—©жҳҜзҪ‘йЎөи¶…ж–Үжң¬еә”з”ЁжҠҖжңҜе·ҘдҪңз»„пјҲWHATWGпјҢ Web HyperText Application Technical Working GroupпјүеңЁWeb Applications 1.0дёӯжҸҗеҮәзҡ„гҖӮиҝҷдёӘи§„иҢғдёӯзҡ„иҚүжЎҲжңҖз»ҲжҲҗдёәдәҶHTML5зҡ„дёҖйғЁеҲҶпјҢеҗҺжқҘеҸҲзӢ¬з«ӢжҲҗдёәиҮӘе·ұзҡ„и§„иҢғгҖӮ

localStorageе’ҢsessionStorageеҜ№иұЎжҳҜWeb Storage APIзҡ„дёҖйғЁеҲҶпјҢд»–д»¬жӢҘжңүзӣёеҗҢзҡ„APIпјҢеҸӘжҳҜеңЁдҪҝз”ЁеңәжҷҜдёҠжңүдәӣдёҚеҗҢпјҡsessionStorage еҜ№иұЎеә”иҜҘдё»иҰҒз”ЁдәҺеӯҳеӮЁеҸӘеңЁдјҡиҜқжңҹй—ҙжңүж•Ҳзҡ„е°Ҹеқ—ж•°жҚ®гҖӮеҰӮжһңйңҖиҰҒи·ЁдјҡиҜқжҢҒд№…еӯҳеӮЁж•°жҚ®пјҢеҸҜд»ҘдҪҝз”Ё globalStorage жҲ– localStorageгҖӮ

дёҺcookieзӣёеҗҢпјҢWeb Storageд№ҹжңүйҷҗеҲ¶пјҢе…·дҪ“зҡ„еӯҳеӮЁйҷҗеҲ¶еҸ–еҶідәҺжөҸи§ҲеҷЁгҖӮдёҖиҲ¬жқҘиҜҙпјҢе®ўжҲ·з«ҜеӯҳеӮЁеӨ§е°ҸжҳҜе’ҢжәҗпјҲеҚҸи®®гҖҒеҹҹгҖҒз«ҜеҸЈпјүзӣёе…ізҡ„пјҢжҜҸдёӘжәҗжңүеӣәе®ҡеӨ§е°Ҹзҡ„еӯҳеӮЁз©әй—ҙгҖӮдёҚеҗҢжөҸи§ҲеҷЁз»ҷlocalStorageе’ҢsessionStroageи®ҫзҪ®дәҶдёҚеҗҢзҡ„з©әй—ҙйҷҗеҲ¶пјҢеӨ§еӨҡж•°йҷҗеҲ¶еңЁ5MBе·ҰеҸіпјҢsafriaеӨ§е°ҸйҷҗеҲ¶еңЁ2.5MBе·ҰеҸіпјҢ еңЁиҝҷйҮҢеҸҜд»ҘжөӢиҜ•web storageеӯҳеӮЁйҷҗеҲ¶гҖӮ

storageе№¶дёҚжҳҜcookieзҡ„жӣҝд»Јж–№жЎҲпјҢеҸӘжҳҜдҪҝз”ЁеңәжҷҜдёҚеҗҢпјҢе®ғжңүд»ҘдёӢеҮ дёӘзӣ®зҡ„пјҡ

  • жҸҗдҫӣжң¬ең°дҝқеӯҳж•°жҚ®зҡ„йҖ”еҫ„пјҢиҖҢдёҚйңҖиҰҒеғҸcookieдёҖж ·жҜҸж¬ЎеҸ‘йҖҒеҲ°жңҚеҠЎз«ҜпјҢж¶ҲйҷӨе®үе…Ёй—®йўҳгҖӮ
  • е…Ғи®ёжң¬ең°дҝқеӯҳжӣҙеӨҡзҡ„ж•°жҚ®гҖӮ
  • е…Ғи®ёдјҡиҜқзә§еҲ«зҡ„ж•°жҚ®еӯҳеӮЁжңәеҲ¶пјҲsessionStorageпјүгҖӮ

Storage вҖӢ

Storageзұ»еһӢдҝқеӯҳзҡ„жҳҜеҗҚ/еҖјеҜ№ж•°жҚ®пјҢе№¶жҸҗдҫӣдәҶд»ҘдёӢж“ҚдҪңAPIгҖӮдёӢйқўзӨәдҫӢlocalStorageиҝӣиЎҢжј”зӨәпјҢдҪҶиҜӯжі•е’ҢSession StorageзӣёеҗҢгҖӮ

  • clear()пјҡеҲ йҷӨжүҖжңүеҖјпјӣдёҚеңЁ Firefox дёӯе®һзҺ°гҖӮ
  • getItem(name)пјҡеҸ–еҫ—з»ҷе®ҡ name зҡ„еҖјгҖӮ
  • key(index)пјҡеҸ–еҫ—з»ҷе®ҡж•°еҖјдҪҚзҪ®зҡ„еҗҚз§°гҖӮ
  • removeItem(name)пјҡеҲ йҷӨз»ҷе®ҡ name зҡ„еҗҚ/еҖјеҜ№гҖӮ
  • setItem(name, value)пјҡи®ҫзҪ®з»ҷе®ҡ name зҡ„еҖјгҖӮ

indexDB вҖӢ

indexDBжҳҜIndex Database APIзҡ„з®Җз§°пјҢе®ғжҳҜжөҸи§ҲеҷЁжҸҗдҫӣзҡ„жң¬ең°ж•°жҚ®еә“гҖӮеҰӮжһңдҪ дәҶи§ЈжңҚеҠЎз«ҜејҖеҸ‘пјҢеҸҜиғҪеҜ№ж•°жҚ®еә“иҝҷдёӘжҰӮеҝөе№¶дёҚйҷҢз”ҹпјҢдҪҝз”Ёе…¶зҡ„дё»иҰҒзӣ®зҡ„дҫҝжҳҜз»“жһ„еҢ–еӯҳеӮЁеүҚз«Ҝж•°жҚ®пјҢж–№дҫҝJavaScriptеҜ№иұЎзҡ„еӯҳеӮЁе’ҢиҺ·еҸ–пјҢеҗҢж—¶д№ҹж”ҜжҢҒжҹҘиҜўе’ҢжҗңзҙўгҖӮжң¬ең°ж•°жҚ®еә“ж„Ҹе‘ізқҖеҸҜд»ҘзҰ»зәҝпјҢ

indexDBзҡ„ж“ҚдҪңйғҪжҳҜејӮжӯҘзҡ„пјҢд»Ҙе…Қйҳ»еЎһеә”з”ЁзЁӢеәҸгҖӮindexDBжңҖеҲқжҳҜеҢ…еҗ«ејӮжӯҘе’ҢеҗҢжӯҘAPIзҡ„пјҢдҪҶжҳҜзӣ®еүҚеҗҢжӯҘAPIе·Ід»Һи§„иҢғдёӯз§»йҷӨдәҶгҖӮ

дёҠйқўеҮ дёӘдәӢд»¶дёӯпјҢevent.targetйғҪжҢҮеҗ‘openRequestпјҢеӣ жӯӨдҪҝз”Ёе“ӘдёӘйғҪеҸҜд»ҘгҖӮdbжҳҜIDBDatabaseзҡ„е®һдҫӢпјҢжҺҘдёӢжқҘе°ұеҸҜдҪҝз”ЁdbиҝӣиЎҢж•°жҚ®еә“зҡ„ж“ҚдҪңгҖӮ

дёәд»Җд№ҲйңҖиҰҒзүҲжң¬еҸ·пјҹ вҖӢ

indexDBе…·жңүеҶ…зҪ®зҡ„вҖңжЁЎејҸпјҲschemeпјүзүҲжң¬жҺ§еҲ¶вҖқжңәеҲ¶пјҢиҝҷеңЁжңҚеҠЎз«Ҝж•°жҚ®еә“дёӯжҳҜдёҚеӯҳеңЁзҡ„гҖӮе’ҢжңҚеҠЎеҷЁж•°жҚ®еә“дёҚеҗҢзҡ„жҳҜпјҢindexDBеӯҳеңЁдәҺе®ўжҲ·з«ҜпјҢе…¶ж•°жҚ®д№ҹеӯҳеңЁдәҺжөҸи§ҲеҷЁдёӯпјҢејҖеҸ‘дәәе‘ҳж— жі•йҡҸж—¶и®ҝй—®е®ғгҖӮеӣ жӯӨпјҢеҰӮжһңжҲ‘д»¬еҸ‘еёғдәҶж–°зүҲжң¬зҡ„еә”з”ЁзЁӢеәҸпјҢе…¶ж•°жҚ®з»“жһ„ж”№еҸҳдәҶпјҢжҲ‘д»¬е°ұйңҖиҰҒжӣҙж–°з”ЁжҲ·жң¬ең°зҡ„ж•°жҚ®з»“жһ„гҖӮ

еҰӮжһңз”Ёжң¬ең°ж•°жҚ®еә“дҪҺдәҺopen()жҢҮе®ҡзҡ„зүҲжң¬еҸ·пјҢupgrageneededдәӢд»¶е°ұи§ҰеҸ‘дәҶпјҢејҖеҸ‘дәәе‘ҳйңҖиҰҒжҜ”иҫғзүҲжң¬еҸ·е№¶еҚҮзә§ж•°жҚ®з»“жһ„гҖӮ

еҰӮжһңж•°жҚ®еә“дёҚеӯҳеңЁж—¶пјҢд№ҹдјҡи§ҰеҸ‘upgradeneededдәӢд»¶пјҢжӯӨж—¶жҲ‘д»¬еҸҜд»Ҙж №жҚ®йңҖиҰҒжү§иЎҢеҲқе§ӢеҢ–е·ҘдҪңгҖӮ

еҜ№иұЎеә“пјҲobject storeпјү вҖӢ

object storeжңүж—¶зҝ»иҜ‘дёәвҖңеҜ№иұЎеӯҳеӮЁвҖқпјҢжңүж—¶еҸҲдјҡиў«зҝ»иҜ‘дёәвҖңеҜ№иұЎд»“еә“вҖқпјҢеңЁиҝҷйҮҢжҲ‘д»¬дҪҝз”ЁвҖңеҜ№иұЎеә“вҖқзҡ„еҗҚз§°гҖӮд№ҹеҸҜд»Ҙиў«з§°дёәвҖңеӯҳеӮЁеҢәвҖқгҖӮ

еҜ№иұЎеә“жҳҜеӯҳеӮЁж•°жҚ®зҡ„еҚ•дёӘеӯҳеӮЁеә“пјҢдҪ еҸҜд»Ҙз®ҖеҚ•зҡ„жҠҠе®ғзҗҶи§Јдёәдј з»ҹе…ізі»еһӢж•°жҚ®еә“дёӯзҡ„иЎЁпјҢдёҖдёӘж•°жҚ®еә“еҸҜд»Ҙе…іиҒ”еӨҡдёӘеҜ№иұЎеә“гҖӮ дёҺе…ізі»ж•°жҚ®еә“дёҚеҗҢзҡ„жҳҜпјҢеҜ№иұЎеә“йҮҢзӣёеҗҢеұһжҖ§еҖјзҡ„зұ»еһӢдёҚйңҖиҰҒдёҖиҮҙпјҲдҫӢеҰӮstoreдёӯеӯҳеӮЁеӨҡдёӘpeopleеҜ№иұЎпјҢе…¶ageеұһжҖ§еҸҜд»ҘжҳҜ23гҖҒвҖңtwenty-fiveвҖқгҖҒundefinedпјүгҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеҲӣе»әеҜ№иұЎеә“ж—¶пјҢеҝ…йЎ»жҢҮе®ҡдёҖдёӘвҖңй”®вҖқжқҘеҢәеҲҶеә“дёӯзҡ„жҜҸдёҖжқЎи®°еҪ•пјҢиҝҷеҸҜиғҪдёҺдј з»ҹж•°жҚ®еә“дёӯзҡ„дё»й”®жҳҜзӣёдјјзҡ„пјҢй”®еҖјдёҚиғҪйҮҚеӨҚпјҢеҗҰеҲҷеӯҳеӮЁж—¶е°ұдјҡжҠҘй”ҷгҖӮжүҖд»ҘжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮй”®жқҘжҗңзҙў/еҲ йҷӨ/жӣҙж–°еҖјгҖӮIndexDBиҝҗиЎҢжҠҠеҜ№иұЎзҡ„дёҖдёӘеұһжҖ§и®ҫзҪ®дёәй”®гҖӮ

  1. еҲӣе»әеҜ№иұЎеә“иҜӯжі•еҰӮдёӢпјҡ

    javascript
    db.createObjectStore(name[, keyOptions]);
    • nameпјҡж–°еҲӣе»әзҡ„еҜ№иұЎеә“еҗҚз§°
    • keyOptionsдёӢйқўдёӨдёӘеұһжҖ§д№ӢдёҖзҡ„еҸҜйҖүеә“еҜ№иұЎ
      • keyPathпјҡеҜ№иұЎеұһжҖ§зҡ„и·Ҝеҫ„пјҢindexDBдҪҝз”ЁиҜҘи·Ҝеҫ„дҪңдёәй”®пјҢеҰӮidгҖӮ
      • autoIncrement: еҰӮжһңдёәtureпјҢеҲҷиҮӘеҠЁз”ҹжҲҗж–°еӯҳеӮЁеҜ№иұЎзҡ„й”®пјҢе®ғжҳҜдёҖдёӘдёҚж–ӯйҖ’еўһзҡ„ж•°еӯ—гҖӮ

  2. еҲ йҷӨеҜ№иұЎеә“иҜӯжі•еҰӮдёӢпјҡ

    javascript
    db.deleteObjectStore(name);
    • nameпјҡиҰҒеҲ йҷӨзҡ„еҜ№иұЎеә“зҡ„еҗҚз§°гҖӮ

дёӢйқўжҳҜдёҖдёӘз®ҖеҚ•зҡ„дҪҝз”ЁзӨәдҫӢпјҡ

javascript
request.onupgradeneeded = event => {
    const db = event.target.result;
    // иҺ·еҸ–еҪ“еүҚиҝһжҺҘж•°жҚ®еә“дёӯзҡ„жүҖжңүеҜ№иұЎеә“еҗҚз§°еҲ—иЎЁпјҢе№¶еҲӨж–ӯжҳҜеҗҰеӯҳеңЁеҗҚз§°дёәвҖңusersвҖқзҡ„еҜ№иұЎеә“
    // еҰӮжһңеӯҳеңЁпјҢеҲҷеҲ йҷӨвҖңusersвҖқеҜ№иұЎеә“
    if (db.objectStoreNames.contains("users")) {
        db.deleteObjectStore("users");
    }
    // еҲӣе»әдёҖдёӘж–°зҡ„вҖңuserеҜ№иұЎеә“вҖқпјҢе№¶жҠҠвҖңusernameвҖқдҪңдёәй”®
    db.createObjectStore("users", { keyPath: "username" });
};

дәӢеҠЎ вҖӢ

еҗҢж ·зҡ„пјҢдәӢеҠЎеҜ№дәҺеұһжҖ§дј з»ҹж•°жҚ®еә“зҡ„ејҖеҸ‘иҖ…д№ҹдёҚдјҡйҷҢз”ҹгҖӮе®ғжҢҮзҡ„жҳҜеҜ№иұЎеә“зҡ„дёҖз»„ж“ҚдҪңпјҢе№¶дё”зЎ®дҝқиҝҷдёҖз»„ж“ҚдҪңе…ЁйғЁжҲҗеҠҹпјҢеҸӘиҰҒдёӯй—ҙжңүдёҖдёӘж“ҚдҪңжү§иЎҢеӨұиҙҘпјҢе°ұе°Ҷд№ӢеүҚзҡ„жүҖжңүж“ҚдҪңйғҪеӣһж»ҡгҖӮ

дҫӢеҰӮпјҡеҪ“жҲ‘д»¬иҙӯд№°дёҖд»¶е•Ҷе“Ғж—¶пјҢжңүдёӨдёӘжӯҘйӘӨпјҡ

  1. д»Һеә“еӯҳдёӯжҠҠиҜҘе•Ҷе“ҒеҮҸ1гҖӮ
  2. жҠҠиҜҘе•Ҷе“ҒеҠ е…ҘеҲ°з”ЁжҲ·зҡ„иҙӯзү©иҪҰдёӯгҖӮ

еҠ е…Ҙз¬¬дёҖдёӘжӯҘйӘӨе·Із»Ҹе®ҢжҲҗпјҢдҪҶжҳҜз¬¬дәҢдёӘжӯҘйӘӨеҚҙеҮәй”ҷдәҶпјҢйӮЈд№Ҳе°ұеҝ…йЎ»еӣһж»ҡз¬¬дёҖдёӘж“ҚдҪңгҖӮ

дәӢеҠЎзҡ„еҲӣе»ә вҖӢ

дәӢеҠЎеҸҜд»ҘеғҸдёӢйқўиҝҷж ·иў«еҲӣе»әгҖӮ

javascript
let transaction = db.transaction(store[пјҢ type]);

  • storeпјҡиҰҒи®ҝй—®зҡ„еҜ№иұЎеә“зҡ„еҗҚз§°

  • typeпјҡдәӢеҠЎзұ»еһӢпјҢд»ҘдёӢзұ»еһӢд№ӢдёҖгҖӮ

    • readonlyпјҡеҸӘиҜ»пјҢй»ҳи®ӨеҖј
    • readwriteпјҡиҜ»еҶҷ

    иҝҳжңүдёҖдёӘversionchangeдәӢеҠЎзұ»еһӢпјҡдҪҶжҳҜиҜҘзұ»еһӢдёҚиғҪиў«жүӢеҠЁеҲӣе»әгҖӮindexDBеңЁиҝһжҺҘж•°жҚ®еә“ж—¶пјҢдјҡиҮӘеҠЁдёәonupgradeneededдәӢд»¶еҲӣе»әдәӢеҠЎгҖӮ

еҰӮжһңдёҚжҢҮе®ҡеҸӮж•°пјҢеҲҷеҜ№ж•°жҚ®еә“дёӯжүҖжңүеҜ№иұЎеә“йғҪе…·жңүеҸӘиҜ»жқғйҷҗгҖӮ

д№ҹеҸҜд»ҘжҢҮе®ҡи®ҝй—®дёҖдёӘжҲ–иҖ…еӨҡдёӘеҜ№иұЎеә“гҖӮ

javascript
// и®ҝй—®еҚ•дёӘеҜ№иұЎеә“
let transaction = db.transaction("users");
// жҲ–иҖ…еҗҢж—¶и®ҝй—®еӨҡдёӘеҜ№иұЎеә“
let transaction = db.transaction(["users", "anotherStore"]);

еҰӮжһңиҰҒдёәеҜ№иұЎеә“еҶҷе…Ҙж•°жҚ®пјҢе°ұйңҖиҰҒдҝ®ж”№дәӢеҠЎзҡ„зұ»еһӢгҖӮ

javascript
let transaction = db.transaction("users", "readwrite");

дәӢеҠЎзҡ„дәӢд»¶ вҖӢ

еҲӣе»әдәӢеҠЎеҗҺпјҢжҲ‘д»¬е°ұеҸҜд»ҘеҜ№еҜ№иұЎеә“жү§иЎҢж·»еҠ /еҲ йҷӨзӯүж“ҚдҪңдәҶпјҢзӨәдҫӢеҰӮдёӢгҖӮ

javascript
let transaction = db.transaction("books", "readwrite"); // (1)

// иҺ·еҸ–еҜ№иұЎеә“иҝӣиЎҢж“ҚдҪң
let books = transaction.objectStore("books"); // (2)

let book = {
    id: "js",
    price: 10,
    created: new Date(),
};

// ж·»еҠ еҜ№иұЎ
let request = books.add(book); 

request.onsuccess = function () {
    console.log("bookиў«ж·»еҠ еҲ°storeдёӯ", request.result);
};

request.onerror = function () {
    console.log("Error", request.error);
};

transaction.oncomplete = function() {
  console.log("дәӢеҠЎжү§иЎҢе®ҢжҲҗ"); // дәӢеҠЎжү§иЎҢе®ҢжҲҗ
};

е’Ңж•°жҚ®еә“дёҖж ·пјҢдәӢеҠЎзҡ„иҜ·жұӮе…·жңүonsuccessе’ҢonerrorдәӢд»¶пјҢеңЁдәӢеҠЎжү§иЎҢжҲҗеҠҹжҲ–иҖ…еӨұиҙҘж—¶иў«и°ғз”ЁгҖӮ

дәӢеҠЎеңЁе®ҢжҲҗд№ӢеҗҺеҸҜд»Ҙзӣ‘еҗ¬oncompleteдәӢд»¶гҖӮ

дәӢд»¶е§”жүҳ вҖӢ

еҰӮжһңиҜҙдәӢеҠЎжү§иЎҢзҡ„жҜҸдёӘиҜ·жұӮйғҪйңҖиҰҒи°ғз”Ё onerror/onsuccessйӮЈе°ҶжҳҜеҫҲйә»зғҰзҡ„пјҢе№ёиҝҗзҡ„жҳҜжҲ‘д»¬еҸҜд»ҘдҪҝз”ЁдәӢд»¶е§”жүҳз»ҷdbжқҘд»ЈжӣҝеӨ„зҗҶгҖӮеҚідҪҝжҳҜIndexDBпјҢжүҖжңүзҡ„дәӢд»¶д№ҹйғҪжҳҜDOMдәӢд»¶пјҢжңүеҶ’жіЎе’Ңе§”жүҳгҖӮ

дёӢйқўзӨәдҫӢжҳҜжңүdb.onerrorеӨ„зҗҶжҚ•иҺ·зҡ„жүҖжңүй”ҷиҜҜгҖӮ

javascript
db.onerror = function (event) {
    let request = event.target; // еҜјиҮҙй”ҷиҜҜзҡ„иҜ·жұӮ

    console.log("Error", request.error);
};

еҰӮжһңдәӢд»¶еңЁеҪ“еүҚдәӢеҠЎиҜ·жұӮе°ұеӨ„зҗҶдәҶпјҢе°ұеҒңжӯўеҶ’жіЎгҖӮ

javascript
request.onerror = function (event) {
    if (request.error.name == "ConstraintError") {
        console.log("Book with such id already exists"); // еӨ„зҗҶй”ҷиҜҜ
        event.preventDefault(); // дёҚиҰҒдёӯжӯўдәӢеҠЎ
        event.stopPropagation(); // дёҚиҰҒи®©й”ҷиҜҜеҶ’жіЎ, еҒңжӯўе®ғзҡ„дј ж’ӯ
    } else {
        // д»Җд№ҲйғҪдёҚеҒҡ
        // дәӢеҠЎе°Ҷдёӯжӯў
        // жҲ‘д»¬еҸҜд»Ҙи§ЈеҶі transaction.onabort дёӯзҡ„й”ҷиҜҜ
    }
};

дәӢеҠЎзҡ„дёӯжӯў вҖӢ

иҰҒжүӢеҠЁдёӯжӯўдәӢеҠЎпјҢеҸҜд»Ҙи°ғз”Ёtransaction.abort()ж–№жі•пјҢеҗҢж—¶transcation.onabortдәӢд»¶д№ҹдјҡиў«и§ҰеҸ‘гҖӮ

javascript
transaction.abort();

дёәд»Җд№ҲдјҡжңүдёҚеҗҢзұ»еһӢзҡ„дәӢеҠЎпјҹ вҖӢ

жҖ§иғҪжҳҜдәӢеҠЎйңҖиҰҒж Үи®°дёә readonly е’Ң readwrite зҡ„еҺҹеӣ гҖӮ

и®ёеӨҡ readonly дәӢеҠЎиғҪеӨҹеҗҢж—¶и®ҝй—®еҗҢдёҖеҜ№иұЎеә“пјҢдҪҶ readwrite дәӢеҠЎдёҚиғҪгҖӮеӣ дёә readwrite дәӢеҠЎдјҡвҖңй”Ғе®ҡвҖқеҜ№иұЎеә“иҝӣиЎҢеҶҷж“ҚдҪңгҖӮдёӢдёҖдёӘдәӢеҠЎеҝ…йЎ»зӯүеҫ…еүҚдёҖдёӘдәӢеҠЎе®ҢжҲҗпјҢжүҚиғҪи®ҝй—®зӣёеҗҢзҡ„еӯҳеӮЁеҢәгҖӮ
